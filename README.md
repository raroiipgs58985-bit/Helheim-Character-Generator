# Character Generator

Запуск:

```bash
python main.py
```

Файлы базы лежат в папке `data/`.

Первая версия умеет:
- читать txt-базы;
- выбирать расу;
- распределять характеристики;
- выбирать родной мир или природу Искина;
- выбирать специальность;
- считать бонусы характеристик;
- считать движение;
- считать БВ;
- считать раны;
- суммировать навыки;
- убирать дубли талантов и возвращать +25 опыта;
- сохранять итоговую карточку в txt.
